### **SAGE Studio**       独立全栈开发      2025/12 - 2026/02
http://studio.sage.org.ai/

*基于前后端分离架构的低代码 AI 工作流可视化平台。项目深度集成 ReactFlow 画布、FastAPI 后端与 LLM 推理测试沙盒，为开发者提供从底层算子编排到上层 Multi-Agent 实时对话的端到端可视化调测环境。*

**主要职能：**
- **核心架构**：主导 SAGE Studio 从 0 到 1 及发布 (PyPI isage-studio)。利用 Vite构建前端，配合 FastAPI 搭建提供 JWT
隔离与 SSE 长连接的多租户后端；通过设计严格的 API 数据契约与系统级 Prompt，高效引导大模型辅助完成了部分复
杂后端逻辑的编码与联调。
- **DAG 拓扑编辑器**：基于 ReactFlow 与 Zustand 构建流编辑器。使用深层状态粒度切分彻底解耦组件，杜绝巨型连线图
渲染卡顿；自研反投影算法实现侧边栏精确拖拽实例化，配合拓扑排序算法 (PipelineBuilder)，将动态节点图实时转译
为SAGE底层算子执行流。
- **流式对话沙盒开发**：开发对话式 Playground，针对超长文本流式响应引发的性能灾难，设计节流缓冲与 reactmarkdown 的增量解析渲染机制。并封装了动态手风琴组件，解决 DOM 重绘抖动，直观展现模型复杂的思维链 (CoT)
及工具调用过程。
- **Agent 编排与 RAG 引擎设计**：在后端设计并整合了 Multi-Agent 调度服务（涵盖意图识别与路由），底层基于
VectorStore 与 KnowledgeManager 封装向量检索栈，打通 6 类异构数据源的清洗、降噪与增量索引链路。

### **FCheck 内容审核平台架构重构**      2026/01 - 2026/03
*面向 UGC 内容安全场景，使用 React 18 从零搭建审核管理平台前端架构，覆盖审核工作台、关键词管理、AI 标注、权限系统等核心模块，针对公司 Vue 3 版本中权限灵活性不足、Token 过期处理粗暴等问题进行了架构级改进。*

**主要职能：**
- **三级权限体系**：设计路由/按钮/平台三级权限模型，配合路由守卫实现动态菜单过滤与页面鉴权；封装 Permission 组
件与 usePermission Hook 支持声明式和编程式两种权限控制，将权限判断从路径字符串匹配升级为权限码匹配，新增
权限配置无需修改过滤逻辑
- **封装 WebSocket 通信层**：实现心跳保活与指数退避自动重连（最大 5 次），通过全局 Store 实时同步任务队列数量与
审核状态变更；消息处理采用 Handler 注册分发机制，新增消息类型无需修改分发逻辑
- **Token 无感刷新**：401 响应时自动触发刷新，并发请求排队等待新 Token 后统一重放，解决原版 Token 过期直接登出
导致用户操作中断的问题
- **虚拟化审核任务列表**：仅渲染可视区域 DOM（万级数据下节点数控制在 30-50 个），支持列表/网格/紧凑三种视图模
式切换，配合 IntersectionObserver 实现无限滚动加载
